# Pokemon Explorer

Application to view Pokemons list with some details such as name, order, type.

## Features

### Implemented Features

- Read Pokemons List via fetch()
- Display pokemon details on grid at start page
- Implement UI elements to met design expectations

### Technology Stack

- **Frontend**: React 19, TypeScript, Vite, Material UI, Tailwind CSS
- **Backend**: PokeApi
- **Database**: No Database used

## Getting Started

To run project type
`npm run dev`

and hit Enter
